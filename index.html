Visualizing JavaScript

This is an Autumn 2012 course for second-year graduate students in New York University’s Interactive Telecommunications Program; taught by me, Stewart Smith, on Fridays, between 15:30 and 18:00 at 721 Broadway (4th Floor) New York NY 10003 in Room 406; course registration call number 17155. Students are expected to be proficient programmers, though JavaScript familiarity is not a prerequisite—nor is graphic design experience.

ITP courses are graded as pass/fail. Auditing is not permitted and the class size will not be augmented to accommodate those wait-listed. Regular attendance, participation in class discussion, and satisfactory completion of assignments is expected.

What’s the content?

Visualizing JavaScript is half coding (JavaScript, of course), half graphic design (visual communication), and the marriage of the two. Course content will be revealed on stewd.io/javascript as the class advances. This site itself is available on GitHub (github.com/stewdio/javascript) for download and comment.
